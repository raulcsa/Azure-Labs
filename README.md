# Microsoft Azure Administrator (AZ-104) - Labs Repository

Este repositorio contiene una colección de laboratorios prácticos diseñados para cubrir todos los objetivos del examen **AZ-104: Microsoft Azure Administrator**. Cada guía detalla paso a paso la configuración y administración de servicios clave en la nube de Azure.

## Propósito del Repositorio

El objetivo de estos laboratorios es proporcionar experiencia práctica en la implementación, administración y supervisión de soluciones de identidad, gobernanza, almacenamiento, computación y redes virtuales en un entorno de Microsoft Azure.

---

## Estructura de los Laboratorios

Los contenidos se dividen en los pilares fundamentales del examen:

### 1. Identidad y Gobernanza
* **Administración de Microsoft Entra ID:** Creación de usuarios internos, invitación de usuarios invitados y gestión de grupos de seguridad.
* **Control de Acceso (RBAC):** Implementación de roles y permisos específicos para recursos.
* **Gobernanza:** Uso de Azure Policy y etiquetas para la organización de recursos.

### 2. Almacenamiento (Storage)
* Configuración de Azure Storage Accounts.
* Gestión de Azure Blob Storage y Azure Files.
* Implementación de seguridad y replicación de datos.

### 3. Recursos de Computación
* Despliegue y administración de Máquinas Virtuales (VMs).
* Configuración de Azure App Services.
* Introducción a Azure Container Instances (ACI) y Azure Kubernetes Service (AKS).

### 4. Redes Virtuales (Networking)
* Configuración de redes virtuales (VNETs) y subredes.
* Implementación de Azure Load Balancer y Application Gateway.
* Conectividad entre redes (VNET Peering) y Network Security Groups (NSG).

### 5. Monitorización y Copias de Seguridad
* Uso de Azure Monitor y Log Analytics.
* Implementación de Backup y recuperación ante desastres (Azure Site Recovery).

---

## Requisitos Previos

Para completar estos laboratorios necesitarás:
* Una suscripción activa de Azure (puedes usar una cuenta gratuita).
* Acceso a [Azure Portal](https://portal.azure.com).
* Conocimientos básicos de conceptos de nube e infraestructura.
